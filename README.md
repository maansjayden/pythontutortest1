# pythontutortest1
# 🇳🇱 Schiphol Customs Check

A quick Python terminal script to practice variables, `if` statements, `f-strings`, and user inputs. Loops are strictly prohibited at this border crossing.

## What's happening here?
* **Inputs:** Taking text from the terminal.
* **Conditionals:** Using `if`, `elif`, and `else` to control the flow.
* **String Methods:** Using `.lower()` to handle case-insensitive passwords.
* **Unit Testing:** Running automated tests to prove the logic holds up.

## How to play
Run the main script in your terminal:
```bash
python main.py
(Hint: Try declaring "easter eggs" and see what customs does to you).

How to run the tests
We wrote a separate test file to make sure the logic in main.py doesn't break if you change things. Run the tests by executing this command in your terminal:

Bash
python test_main.py
If everything is correct, it will say OK at the bottom.

Your test: Try breaking the logic in main.py on purpose (like changing the passcode), then run python test_main.py again to see the tests catch your mistake!
